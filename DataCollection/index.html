Mean

In a traditional experiment, the mean is the null value. That is, the value the experiment would result in if the hypothesis is incorrect. In an experiment that is intentionally producing misleading results, this is the actual value that a reliable experiment would resulte in.

Standard Deviation

This is a value based on the variance of a dataset. In a traditional experiment this would be calculated based on the data that is collected, however, given that the data here is being randomly generated, the standard deviation needs to be entered so that the random generation is based on something. If this is larger, the statistically significant values will be closer to the mean.

Population Size

This is the number of subjects that are generated in every trial. If this is larger, the statistically significant values will be closer to the mean.

Trials

This is the number of trials that will be run. If this is larger, there will be a greater chance that one of the trials will be statistically significant.

Statistical Significance

P-Value is the meassure of the likely hood that a result is not the null case (value if the hypothesis is not correct) being skewed by random chance. A statistically significant is usually considered to be a result that has a p-value of 5%. This means that there is a 5% chance that the true value is the null case.
